## Default Permission

Default permissions for the plugin

- `allow-adCreate`
- `allow-adHide`
- `allow-adIsLoaded`
- `allow-adLoad`
- `allow-adShow`
- `allow-configRequest`
- `allow-configure`
- `allow-requestTrackingAuthorization`
- `allow-trackingAuthorizationStatus`
